🔥 Firebase 설정 가이드

애슬리트 타임을 실제 서비스로 만들기 위한 Firebase 연동 가이드

예상 소요 시간: 15-20분

왜 Firebase가 필요한가요?

❌ 현재 상태 (localStorage)

  • • 각자 브라우저에만 데이터 저장
  • • 다른 사용자와 공유 불가능
  • • 내가 쓴 글을 다른 사람이 못 봄
  • • 진짜 커뮤니티 기능 불가

✅ Firebase 연동 후

  • • 실시간 데이터 동기화
  • • 모든 사용자가 같은 데이터 공유
  • • 실시간 채팅 가능
  • • 진짜 커뮤니티 서비스!
1

Firebase Console 접속

Google 계정으로 Firebase Console에 로그인합니다.

Firebase Console 열기
2

새 프로젝트 생성

  1. 1. "프로젝트 만들기" 클릭
  2. 2. 프로젝트 이름: athletetime (또는 원하는 이름)
  3. 3. Google Analytics: 사용 안함 선택 (나중에 추가 가능)
  4. 4. "프로젝트 만들기" 클릭
3

웹 앱 추가

  1. 1. 프로젝트 개요 페이지에서 웹 아이콘 클릭
  2. 2. 앱 닉네임: 애슬리트 타임 웹
  3. 3. "Firebase 호스팅 설정" 체크 (선택사항)
  4. 4. "앱 등록" 클릭
4

Firebase 설정값 복사

표시되는 설정값을 복사합니다:

const firebaseConfig = {
apiKey: "AIzaSy...",
authDomain: "athletetime.firebaseapp.com",
databaseURL: "https://athletetime-default-rtdb.firebaseio.com",
projectId: "athletetime",
storageBucket: "athletetime.appspot.com",
messagingSenderId: "123456789",
appId: "1:123456789:web:abc..."
};
5

Realtime Database 활성화

  1. 1. 왼쪽 메뉴에서 "빌드" → "Realtime Database" 클릭
  2. 2. "데이터베이스 만들기" 클릭
  3. 3. 위치: 미국(us-central1) 선택
  4. 4. 보안 규칙: 테스트 모드에서 시작 선택
  5. 5. "사용 설정" 클릭

중요: 테스트 모드는 30일간만 유효합니다. 정식 운영 시 보안 규칙을 설정해야 합니다.

6

Storage 활성화

  1. 1. 왼쪽 메뉴에서 "빌드" → "Storage" 클릭
  2. 2. "시작하기" 클릭
  3. 3. 보안 규칙: 테스트 모드에서 시작 선택
  4. 4. 위치: 가장 가까운 지역 선택
  5. 5. "완료" 클릭
7

코드에 적용하기

복사한 설정값을 프로젝트에 적용합니다:

1. HTML 파일에 Firebase SDK 추가

<!-- Firebase SDK -->
<script src="https://www.gstatic.com/firebasejs/9.22.0/firebase-app-compat.js"></script>
<script src="https://www.gstatic.com/firebasejs/9.22.0/firebase-database-compat.js"></script>
<script src="https://www.gstatic.com/firebasejs/9.22.0/firebase-storage-compat.js"></script>

2. firebase-config.js 파일 수정

// 복사한 설정값으로 교체
const firebaseConfig = {
apiKey: "여기에_실제_값_입력",
authDomain: "여기에_실제_값_입력",
// ... 나머지 설정값
};
8

테스트 및 확인

  1. 1. 브라우저에서 애슬리트 타임 열기
  2. 2. 개발자 도구 콘솔 확인 (F12)
  3. 3. "Firebase initialized successfully" 메시지 확인
  4. 4. 게시글 작성 및 채팅 테스트
  5. 5. 다른 브라우저/기기에서 동일한 데이터 확인

문제 해결

❓ Firebase not initialized 오류

  • • Firebase SDK 스크립트가 제대로 로드되었는지 확인
  • • firebase-config.js의 설정값이 올바른지 확인
  • • 네트워크 연결 상태 확인

❓ Permission denied 오류

  • • Firebase Console에서 Database 규칙이 테스트 모드인지 확인
  • • 테스트 모드 만료 여부 확인 (30일)

❓ 데이터가 동기화되지 않음

  • • 인터넷 연결 상태 확인
  • • Firebase Console에서 Database URL이 올바른지 확인
  • • 브라우저 캐시 삭제 후 재시도
