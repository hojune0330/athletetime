URL 자동 링크 기능

✨ 자동 링크 변환

게시물과 댓글에 포함된 웹 주소가 자동으로 클릭 가능한 링크로 변환됩니다!

지원하는 URL 형식

  • ✅ https://example.com
  • ✅ http://example.com
  • ✅ www.example.com
  • ✅ example.com (주요 도메인)
  • ✅ user@example.com (이메일)

🧪 테스트 예시

일반 웹사이트

작성한 내용:

오늘 훈련 영상 올렸어요! https://www.youtube.com/watch?v=example

표시되는 모습:

오늘 훈련 영상 올렸어요! youtube.com/watch?v=example

소셜 미디어 링크

작성한 내용:

제 인스타 구경오세요 instagram.com/athlete_kim 트위터도 있어요 twitter.com/athlete_kim

표시되는 모습:

제 인스타 구경오세요 instagram.com/athlete_kim
트위터도 있어요 twitter.com/athlete_kim

이메일 주소

작성한 내용:

문의사항은 contact@athlete-time.com 으로 보내주세요

표시되는 모습:

문의사항은 contact@athlete-time.com 으로 보내주세요

여러 개의 링크

작성한 내용:

홈페이지: www.example.com
유튜브: https://youtube.com/channel/abc
깃허브: github.com/username
네이버: naver.com

표시되는 모습:

홈페이지: www.example.com

유튜브: youtube.com/channel/abc

깃허브: github.com/username

네이버: naver.com N

사용 팁

  • • URL을 입력하면 자동으로 링크로 변환됩니다
  • • 링크 클릭 시 새 탭에서 열립니다
  • • 긴 URL은 자동으로 축약되어 표시됩니다
  • • 주요 SNS는 전용 아이콘이 표시됩니다
  • • 이메일 주소는 클릭 시 메일 앱이 열립니다
