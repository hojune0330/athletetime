📸 이미지 업로드 가이드

새로운 대용량 이미지 자동 압축 기능!

  • ✅ 최대 50MB까지 업로드 가능 (이전: 5MB)
  • ✅ 자동 스마트 압축 - 품질 유지하면서 용량 최적화
  • ✅ 압축 진행 상태 표시 - 실시간 처리 과정 확인
  • ✅ 압축률 표시 - 얼마나 압축되었는지 확인

스마트 압축 전략

5-10MB

압축 후에도 5MB를 초과하면 자동으로 품질과 크기를 추가 조정합니다.

사용 방법

  1. 1
    파일 선택
    최대 3장, 각 50MB까지 가능
  2. 2
    자동 압축
    대용량 파일은 자동으로 압축 처리
  3. 3
    압축 결과 확인
    압축률과 최종 용량 표시
  4. 4
    게시물 작성
    압축된 이미지와 함께 게시

압축 예시

📱 스마트폰 원본 사진 (4000x3000) 12MB → 1.8MB (85% 압축)
📷 DSLR 고화질 사진 (6000x4000) 25MB → 2.3MB (91% 압축)
🖼️ 스크린샷 (2560x1440) 8MB → 1.2MB (85% 압축)

유용한 팁

  • 압축 후에도 품질이 걱정되면 원본을 별도로 보관하세요
  • 여러 장을 한 번에 선택하면 더 빠르게 처리됩니다
  • 압축 중에는 잠시 기다려주세요 (대용량일수록 시간 소요)
  • 3일 후 원본이 자동 삭제되니 중요한 사진은 미리 다운로드하세요
