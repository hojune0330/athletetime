✅ 모바일 개선 사항

  • 고정 첫번째 열: km 페이스 열이 스크롤 시에도 고정
  • 스크롤 표시기: "좌우 스크롤" 안내 배지 추가
  • 터치 최적화: 버튼과 텍스트 크기 증가
  • 다운로드 버튼: 차트 하단으로 이동, 전체 너비
  • 3000m 장애물: 세로 배치로 변경

📱 모바일 (≤768px)

  • • 글자 크기: 11px
  • • 패딩: 6px
  • • 첫 열 고정
  • • 스크롤 안내
  • • 다운로드 버튼 재배치

💻 데스크톱 (>1024px)

  • • 기존 레이아웃 유지
  • • 모든 기능 정상 작동
  • • 넓은 테이블 표시
  • • 다운로드 버튼 우측 상단
  • • 워터마크 제거 상태

🎯 핵심 개선 내용

1. Sticky 첫번째 열:

테이블을 좌우로 스크롤해도 km 페이스 열은 항상 보임

2. 스크롤 인디케이터:

스크롤 가능한 테이블에 "좌우 스크롤" 배지 표시

3. 터치 친화적:

모든 터치 영역 최소 44px 이상 확보

/* 첫번째 열 고정 */
.pace-table thead th:first-child,
.pace-table tbody td:first-child {

  left: 0;

  z-index: 10;
