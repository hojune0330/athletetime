검증 피드백 구현 테스트

✅ 구현 완료 항목

  • ✓ 페이스 범위 표시 (단일값 → 최소/최대 범위)
  • ✓ RPE (운동자각도) 척도 추가 (3-4/10 ~ 10/10)
  • ✓ 15개 이상의 표준 운동 템플릿 구현
  • ✓ 환경 조건 조정 (보통/추움/더움/고온다습)
  • ✓ 탭 인터페이스 (표준 템플릿 / 맞춤 훈련)
  • ✓ 용어 수정 (정확성 → 정밀도)

📊 페이스 범위 예시

Easy: 5:30-4:30/km (RPE 3-4/10)
Marathon: 4:20-3:50/km (RPE 5-6/10)
Threshold: 3:45-3:30/km (RPE 7-8/10)
Interval: 3:20-3:10/km (RPE 9/10)
Repetition: 3:00-2:50/km (RPE 10/10)

총 15개+

🌡️ 환경 조정

• 보통 (15-20°C): 조정 없음
• 추운 날씨 (<10°C): 페이스 +2%
• 더운 날씨 (>25°C): 페이스 +4%
• 고온다습 (>25°C, 습도 >70%): 페이스 +6%, 강도 -10%

테스트 방법:
1. 훈련 계산기 열기
2. 기록 입력 (예: 5km 20:00)
3. 페이스 범위 확인
4. "운동 세부사항" 섹션에서 템플릿 확인
5. 환경 조건 변경하여 실시간 조정 확인
