페이스 계산기 버전 테스트

현재 구조:

  • index.html - 메인 랜딩 페이지 (원본) 완료
  • pace-calculator-selector.html - 버전 선택 페이지 완료
  • pace-calculator.html - PC 버전 완료
  • pace-calculator-mobile.html - 모바일 버전 완료

주요 개선사항:

  • ✅ 워터마크 제거 및 푸터 크레딧 추가
  • ✅ html2canvas로 다운로드 기능 수정
  • ✅ 모바일 테이블 첫 열 고정 (sticky)
  • ✅ 터치 친화적인 큰 버튼
  • ✅ 스크롤 인디케이터
  • ✅ 7일간 버전 선택 저장
메인 페이지로 이동 버전 선택 페이지로 이동 모바일 버전 직접 열기 PC 버전 직접 열기

테스트 방법:

  1. 메인 페이지에서 "페이스 계산기 & 차트" 클릭
  2. 버전 선택 화면에서 원하는 버전 선택
  3. 선택한 버전이 7일간 기억됨
  4. 각 버전에서 PNG/PDF 다운로드 테스트
