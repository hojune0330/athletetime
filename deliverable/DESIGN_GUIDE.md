# 카드뉴스 디자인 가이드라인

기준 샘플: 남자 100m 결선 카드뉴스 (2026 전국체전)

## 확정된 디자인 원칙

### 전체 톤
- 화이트 배경 기반 미니멀 디자인
- 불필요한 장식 요소 없음 (그라데이션, 아이콘, 배경 패턴 없음)
- 정보 밀도가 높되 여백으로 숨을 쉬는 레이아웃
- 느낌: "스포츠 매거진의 데이터 페이지"

### 레이아웃 구조 (위에서 아래)
1. 상단 헤더: 대회명 (작은 글씨) + 종목명 (큰 볼드)
2. 구분선
3. 테이블 헤더: # / 선수 / 소속 / 기록 / PB
4. 결과 행: 8명까지 표시
5. 구분선
6. 하단 푸터: 날짜+장소 | 바람 정보(트랙만) | 브랜딩(ATHLETIC NEWS)

### 타이포그래피
- 종목명: 48px, font-weight 900, 블랙
- 대회명: 15px, font-weight 500, 그레이
- 선수 이름: 20px, font-weight 700, 블랙
- 소속: 15px, font-weight 400, 그레이
- 기록: 22px, font-weight 700, tabular-nums
- PB: 13px, font-weight 400, 라이트 그레이
- 폰트: Noto Sans CJK KR (시스템 폰트 우선)

### 색상 팔레트
- 배경: #FFFFFF
- 주요 텍스트: #111111
- 보조 텍스트: #666666
- 약한 텍스트: #999999
- 1위 기록 강조: #03C75A (네이버 그린)
- 구분선(굵은): #111111
- 구분선(행 사이): #E5E5E5

### 1위 강조 방식
- 1위 기록 텍스트 색상만 그린으로 변경
- 배경색 변경 없음, 아이콘 없음, 메달 이모지 없음
- 절제된 강조가 핵심

### 여백 규칙
- 카드 크기: 1080x1080 (인스타그램 피드)
- 좌우 패딩: 80px
- 상단 패딩: 72px
- 하단 패딩: 60px
- 헤더-테이블 간격: 36px (구분선 포함)

### 하지 말 것
- 메달 아이콘 사용 금지
- 배경색 번갈아 칠하기 (zebra striping) 금지
- 그라데이션, 그림자 효과 금지
- 과도한 볼드/색상 금지
- 로고 이미지 크게 넣기 금지

|-----------|----------|----------|--------------|
| 트랙 (단거리) | 기록 | O | 10.25 |

### 바람 표시 규칙
- 200m 이하 트랙 종목: 표시
- 멀리뛰기, 세단뛰기: 표시
- 그 외: 표시하지 않음
