페이스 계산기 다운로드 기능 개선 사항

✨ 주요 개선 내용

  • 워터마크 가시성 향상: 다운로드시 opacity 0.5로 증가 (기존 0.25)
  • 텍스트 간격 문제 해결: 요소 복제 방식으로 렌더링 문제 방지
  • 고품질 렌더링: 2배 스케일로 선명한 이미지 생성
  • 파일명 개선: "AthleteTime_장호준코치제작_" 접두사 추가
  • 3000m 장애물 섹션: 내측/외측 구성 동시 표시

🎨 워터마크 스타일링

  • • 기본 상태: opacity 0.15 (은은하게)
  • • 다운로드시: opacity 0.5 (선명하게)
  • • 폰트 굵기: 600-800 (bold)
  • • 텍스트 그림자 추가

📐 레이아웃 개선

  • • table-layout: fixed 적용
  • • border-spacing 최적화
  • • line-height: 1.2 설정
  • • vertical-align: middle

📥 다운로드 프로세스

  1. 원본 요소를 복제하여 화면 밖에 배치
  2. 다운로드 버튼 제거 및 워터마크 강조
  3. dom-to-image로 고품질 PNG 생성
  4. 복제된 요소 제거 및 원본 복원
개선된 페이스 계산기 열기 3000m 장애물 섹션으로 이동

테스트 방법: 페이스 계산기를 열고 각 차트의 PNG/PDF 다운로드 버튼을 클릭하여 워터마크 가시성과 텍스트 간격이 올바르게 표시되는지 확인하세요.
