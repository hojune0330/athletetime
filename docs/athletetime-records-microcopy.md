# AthleteTime — /records UX Microcopy Pack (3차)

작성: Claude(신뢰/문과 영역) · 기준 커밋 `68de3ac` · **이 문서는 카피 제안만 — 코드 미수정.**

> **이번 단계의 목표**
> 1·2차에서 정리한 "긴 법적 방어 문장"을 **토스/구글처럼 짧고 담백한 신뢰 문구**로 줄인다.
> 화면은 가볍게, 신뢰 장치는 분명하게. 한 화면에 긴 고지문을 늘어놓지 않는다.
>
> **원칙**
> - 한 문장 = 한 메시지. 가능하면 **15자 안팎**, 길어도 한 줄.
> - 친근한 수다체보다 **담백한 UX 문장**을 우선한다.
> - "법적으로 안전하다" 같은 **단정은 쓰지 않는다.**
> - person_no·생년월일·원본 식별자를 저장/노출한다는 전제를 **절대 깔지 않는다.**
> - 적용 위치는 Codex가 지정 → 본 문서는 후보 문구만 제공.

산출 형식: `| 화면 | 상황 | 권장 문구 | 피해야 할 문구 | 이유 |`
