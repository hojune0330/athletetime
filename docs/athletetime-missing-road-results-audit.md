작성일: 2026-07-07

## 목적

예천 도효자배처럼 파일명이 `종합기록지.pdf`로만 올라오거나, 결과파일 없이 뉴스/상세 페이지에 흩어진 로드대회 결과를 다시 찾기 위한 감사 기록이다.

## 이번 확인: 예천 도효자배

- 기존 정본 데이터(`data/results/*.json`)에는 도효자배 대회 결과가 정규 대회로 들어와 있지 않았다.
- 기존 파일명 기반 후보 발견은 2025년처럼 파일명이 일반적인 경우(`종합기록지20250312.pdf`)를 놓칠 수 있었다.
- 새 감사 도구는 KAAF 연간 국내경기 표의 같은 행에서 `대회명 + 기간 + 장소 + FILEs_4 결과첨부`를 함께 읽는다.

## KAAF 외 보강 후보

- 2014년 1회 대회는 예천뉴스 기사에서 결과 요약이 확인된다. 다만 기사 본문 전체를 복제하지 않고, 사실 데이터만 별도 검수 후 보강한다.
- 2017년 3회 대회는 지역 언론에 결과 요약이 확인된다. 공식 첨부가 없으므로 보조 출처로만 사용한다.
- 2016년 2회와 2019년 5회는 KAAF 행은 있으나 결과첨부가 없어, 한국중고육상연맹/지역 언론/학교 보도자료를 추가 탐색해야 한다.

## 재실행 방법

```bash
node tools/audit-kaaf-competition-results.js \
  --query 도효자 \
  --from-year 2000 \
  --to-year 2026 \
  --output .ultra/docs/research/kaaf-dohyoja-audit-2000-2026-20260707.json \
  --report .ultra/docs/research/kaaf-dohyoja-audit-2000-2026-20260707.md \
  --json
```

## 운영 원칙

- `result.kaaf.or.kr` 같은 차단 표면은 신규 수집 경로로 사용하지 않는다.
- KAAF 공개 국내경기 페이지와 `FILEs_4` 결과첨부를 1차 공식 원천으로 둔다.
- 파일명이 애매할 때는 파일명만 보지 않고 같은 행의 대회명을 함께 저장한다.
- 원본 파일은 Git에 올리지 않는다. PR에는 도구, manifest, hash, 보고서만 남긴다.
- 뉴스/지역 보도는 공식 첨부가 없을 때의 보조 출처이며, 원문을 복제하지 않고 사실 데이터만 검수해서 사용한다.
