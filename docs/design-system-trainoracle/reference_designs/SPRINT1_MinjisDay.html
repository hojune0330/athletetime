4 Systems · 한 사용자의 하루에 어떻게 짜이는가

SYSTEM 1

Identity

"이 코치는 나를 안다"는 신호. 호명, 메모리, 시그니처, 대화 누적.

  • "민지님" / "민지" 컨텍스트별 호명
  • 장호준 AI 일관 페르소나
  • Memory chip · 컨텍스트 항상 표시
  • Personal Archive · 시간 누적
  • 1년 전 오늘 · 회상 모듈
SYSTEM 2

Visualization

훈련 분류 즉시 인식. Tier 4단계 (strong/mid/subtle/wash).

  • 에너지 시스템 6 × 4 tier = 24 색상
  • Strong tier · 카드 좌측 5px
  • Comparison mode · vs 지난 사이클
  • 모든 통계 drill-down 가능
  • 색 + 코드 + 패턴 3중 인코딩
SYSTEM 3

Trust

4 Layer 검증. AI는 항상 자기 점수를 공개한다.

  • Verdict + Confidence (필수)
  • 인용 4 tier (J/P/Y/C)
  • Process trace · 본 데이터 공개
  • Track record · 동의율/일치율
  • Disagreement · 반대 의견 의무
SYSTEM 4

Feedback

6단계 시간 tier + 5 모드. 살아있는 코치.

  • Realtime · Daily · Weekly · Cycle · Monthly · Season
  • Mode 5종 · Normal/Caution/Recovery/Race/Off
  • 자동 mode 전환 (통증 → CAUTION)
  • 알림 규칙 · 무의미 알림 X
  • Auto-narrative · 모든 통계 옆 1줄
